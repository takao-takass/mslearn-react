このプロジェクトの生成には、次のツールが使用されました:
- create-vite

このプロジェクトを生成するのに次の手順が使用されました:
- create-vite を使用して React プロジェクトを作成します: `npm init --yes vite@latest mslearn-react -- --template=react`.
- プロジェクト ファイル (`mslearn-react.esproj`) を作成します。
- `launch.json` を作成してデバッグを有効にします。
- プロジェクトをソリューションに追加します。
- このファイルを書き込みます。
